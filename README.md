# WmCreate

## 概述：

武汉美萌工程化开发脚手架工具, 目的是为了快捷处理后台开发及Taro小程序高度耦合且重复性极高的工作内容.

## 特征:

### ant-design-pro-components:

- Generate ProTable  Props:

  使用:  

  ```sh
  wm create -t
  ```

  |               功能                | 是否可用 |
  | :-------------------------------: | :------: |
  | 根据swaggerJson过滤出所有列表接口 |    ✅     |
  |       生成ProTable Columns        |    ✅     |
  |     手动排除Columns业务无关项     |    🚧     |
  |       手动对Columns进行排序       |    🚧     |

## 安装:

``` sh
npm install wm-create@latest -g
```
